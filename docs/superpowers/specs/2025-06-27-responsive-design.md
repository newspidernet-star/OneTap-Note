# Smart Scribe — 响应式三档布局设计

日期: 2025-06-26

## 概述

将 Workstation 从固定宽度桌面三栏改为响应式布局，支持桌面 / 平板 / 手机三档。

## 断点策略

| 断点 | 宽度 | 布局 |
|------|------|------|
| 桌面 `lg` | ≥1024px | 三栏并排：上传 220px ↔ AI总结 flex-1 ↔ 时间线 400px（可折叠） |
| 平板 `md` | 768-1023px | 两栏：上传 200px + AI总结，时间线收起到侧边标签 |
| 手机 `sm` | <768px | 底部三 Tab 图标栏，单面板全宽 |

## 手机导航

- 底部 `fixed bottom-0` 吸底栏，三个图标按钮：云上传 / AI总结 / 时间线
- state: `tabView: "upload" | "summary" | "timeline"`，仅 `<768px` 生效
- 桌面/平板上 `tabView` 忽略，三栏/两栏正常显示
- 手机 Tab 栏高度约 56px，底部留 safe-area-inset-bottom

## 组件适配

| 组件 | 桌面 | 平板 | 手机 |
|------|------|------|------|
| Header 标题栏 | 品牌 + 标题输入 + 状态 + 设置 | 品牌 + 标题输入（窄） + 设置 | 品牌 + 设置（标题输入隐藏） |
| 左侧上传栏 | 220px 固定 | 200px 固定 | 全宽（tab=upload 可见） |
| 中栏 AI 总结 | flex-1 | flex-1 | 全宽（tab=summary 可见） |
| 右栏时间线 | 400px 可折叠 | 折叠态竖标签 | 全宽（tab=timeline 可见） |
| IslandButton | w-[320px] | w-[280px] | w-full max-w-[280px] |
| 删除确认弹窗 | 居中卡片 | 居中卡片 | 底部 sheet |
| 上传区域 | 正常 | 正常 | 正常 |
| 链接输入 | 正常 | 正常 | 按钮到输入框下方（已修） |

## Tailwind 前缀映射

- 桌面样式用默认 class（无前缀）
- 平板覆盖用 `md:` 前缀
- 手机覆盖用 `max-md:` 前缀（或 `sm:` 配合移动优先）

## 文件改动

- `frontend/src/pages/Workstation.tsx`：主文件，所有三栏布局 + tabView state + Tailwind 响应式前缀
- 不新增组件文件——靠 Tailwind 和现有结构

## 副作用

- 三栏间分隔线在手机端不显示（竖线变单栏无意义）
- 历史会话列表在手机上可能变折叠或省略
- Header 标题栏高度在手机上可能微调
