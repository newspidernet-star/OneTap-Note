# Smart Scribe — 手机端一页流 + 吸顶播放器设计

日期: 2025-06-27

## 概述

手机端从"三 Tab 切换"改为"单页滚动 + 吸顶播放器"，桌面端不变。

## 手机端布局（<768px）

```
┌─────────────────────────┐
│ ☰ Smart Scribe       ⚙ │  ← 标题栏：汉堡菜单 + 品牌 + 设置
├─────────────────────────┤
│                         │
│  📤 上传媒体            │  ← 上传区（空/新会话可见，已有会话隐藏）
│  ┌──────────────────┐   │
│  │ 粘贴链接...       │   │
│  └──────────────────┘   │
│  [添加链接]             │
│                         │
│  ┌──────────────────┐   │
│  │ ✨ 生成 AI 总结   │   │  ← IslandButton（可点）
│  └──────────────────┘   │
│                         │
│  ═══ 吸顶播放器 ══════  │  ← 处理完后 IslandButton 变媒体播放器 + sticky top-0
│  [视频播放 / 音频播放]   │     滚过顶部就吸住，回滚复原
│                         │
│  ⭐ 核心要点            │  ← AI 总结
│  1. 法拉第...           │
│  2. 楞次定律...         │
│                         │
│  ▶ 原文证据 · 3 块      │  ← 折叠卡片，点开显示时间线
│     [证据块列表...]      │
│                         │
└─────────────────────────┘
```

## 汉堡菜单

- 点击 ☰ 弹出 Drawer/Sheet 覆盖层
- 内含：历史会话列表（每项可点选） + 「+ 新建会话」按钮
- 会话语录 hover 显示 × 删除

## 吸顶播放器逻辑

- IslandButton 媒体预览模式（mode="preview"）时给容器加 `sticky top-0 z-10`
- 只在此模式 sticky，button/pipeline/error 模式不用
- 滚动超过 IslandButton 原始位置 → 吸在屏幕顶部
- 回滚到原始位置 → 回归原位
- 点时间戳 → 页面自动滚到 IslandButton，视频跳转

## 已有会话展示

- 选已有已完成会话：上传区隐藏、IslandButton 变预览模式、直接显示总结
- 选空/新会话：上传区显示、IslandButton 按钮模式

## 改动文件

- `frontend/src/pages/Workstation.tsx`：去掉底部 Tab 栏、加汉堡菜单、改手机单页布局、已有/新会话两态切换
- `frontend/src/components/IslandButton.tsx`：加 sticky 模式（仅 preview 时）
- 去掉 `tabView` 状态及其相关 JSX
